독립성 검정(Test for Independence)

예를 들어 근속년수과 직업만족도 간의 관계 또는 성별에 따른 부서선호도의 관련성을 알기 위해 실시하는 검정이다. 두 요인 간에 관련성이 있는지 또는 없는지, 즉 서로 독립인지 아닌지를 검정한다. 가설은 다음과 같다.

귀무가설 H0 : 요인 A, B가 서로 독립이다.
대립가설 H1 : H0가 아니다.

유의수준이 5%일 경우 계산된 P-value가 0.05이하이면 귀무가설을 기각한다. 즉 두 요인 A와 B는 서로 독립이 아니라는 결론을 내린다.



1. MINITAB을 사용한 독립성 검정 (데이터 : Hypo06.mtw)

회사 내의 성별과 부서 선호도 간의 관련성을 조사하기 위하여 100명을 임의로 추출 하여 다음의 데이터를 얻었다. 그러면 이 자료를 이용하여 성별과 부서 선호도가 서로 영향을 미치고 있는지 a = 0.05로 검정하라.


귀무가설 H0 : 성별과 부서 선호도는 서로 영향을 미치지 않는다. 즉, 서로 독립이다.
대립가설 H1 : H0가 아니다.

MINITAB Worksheet에 아래의 데이터를 입력한다




1.2 MINITAB의 실행

① Stat > Tables > Cross Tabulations 선택





② Columns containing the table 란에 '남'과 '여'를 입력한다.




③ OK버튼을 클릭한다.

[ Session 창에 나타난 결과 ]


관측돗수와 귀무가설 하에서 기대돗수가 계산되었으며 카이제곱 통계량은 2.262이다. 그리고 P-Value가 0.323이므로 H0를 기각할 수 없다. 즉, 성별은 부서 선호도에 영향을 미치지 않는다고 말할 수 있다.



동일성 검정(Test for Homogeneity)

같은 제품을 생산하는 여러 대의 기계의 불량률이 동일할까? 등과 같이 여러 모집단 간의 분포가 동일한 지를 검정하고자 한다.

귀무가설 H0 : 각 모집단간 분포가 동일하다.
대립가설 H1 : H0가 아니다.

동일성 검정의 자료구조는 독립성 검정의 자료구조와 형태가 동일하며 검정 결과도 역시 같다. 그러므로 MINITAB에서 동일성 검정은 독립성 검정과 동일한 메뉴를 사용한다. 따라서 유의수준이 5%일 경우 P-value가 0.05이하이면 귀무가설을 기각한다. 즉 모집단 간의 확률분포가 다르다는 결론을 내린다.



1. MINITAB의 Cross Tabulation을 사용한 동일성 검정 (데이터 : Hypo07.mtw)

어느 공장에서 A, B, C 3 대의 기계로 생산된 제품을 기계별로 각각 100개씩 샘플을 뽑아 양/불량을 검사한 결과를 담은 현장 데이터(Raw Data)를 분할표로 정리한 결과가 다음 표이다. 이 현장 데이터로부터 직접 기계별로 불량률이 서로 다르다고 할 수 있는지 a = 0.05로 검정하고자 한다.


귀무가설 H0 : 기계 A, B, C의 불량률은 모두 동일하다.
대립가설 H1 : H0가 아니다.



1.1 데이터의 입력

MINITAB Worksheet에 아래의 데이터를 입력한다.




1.2 MINITAB의 실행

① Stat > Tables > Cross Tabulations 선택




② Classification variables 란에 '기계'와 '판정결과'를 입력한다. Chi-Square analysis를 체크
    하고 돗수와 기대돗수를 표시하기 위해 Above and expected count를 체크한다.




③ OK버튼을 클릭한다.

[ Session 창에 나타난 결과 ]


관측돗수와 귀무가설 하에서 기대돗수가 계산되었으며 카이제곱 통계량은 3.116이다. 그리고 P-Value가 0.211이므로 H0를 기각할 수 없다. 즉, 기계별로 불량률이 다르다고 할 수 없다.
