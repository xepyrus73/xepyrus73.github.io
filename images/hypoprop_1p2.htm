모비율의 검정
Stat>Basic Statistics>1 Proportion


불량률, 찬성률, 실업률 등과 같이 모집단에서 어떤 특정 속성을 갖는 것의 비율을 모비율이라 말한다. 모비율 p에 관한 검정과 모비율의 신뢰구간을 구하기 위해 1-Proportion을 사용한다. n이 충분히 큰 경우 ( np0 > 5 )일 때는 정규분포를 따른다는 가정을 하고 Options 메뉴의 Use test interval based on normal distribution 란에 체크를 하게 된다.
