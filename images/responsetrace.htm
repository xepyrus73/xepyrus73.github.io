혼합물 실험의 Response Trace Plot



분석을 통하여 얻어진 결과를 토대로 최적의 값을 구하는 최적수준을 구하는 것이 최종의 목적이다. 어떤 수준의 조합인 경우에 최적 값을 갖는지에 대하여 MINITAB에서는 Response Trace Plot, Contour/Surface Plot, Response Optimizer 등의 방법을 제공한다.


1. Response Trace Plot의 작성 (데이터 : Sim_Cen_r.mtw)

 이 Plot은 각각의 성분들이 반응변수에 어떻게 영향을 주는지에 대해 분석할 수 있다.

① Stat > DOE > Response Surface > Contour/Surface(Wireframe)Plots...을 선택한다.



② Response 란에 '탄성계수'를 입력한다.
Trace Direction<1>에서 탄성계수에 영향을 미치는 성분 중에서 만약 Pseudo 성분이 있는 경우에는 Piepel을 없는 경우에는 Cox를 선택하여 Plot을 그린다.
Model Fitted In<2>에서 모델의 적합 값을 구할 때 비율(Proportions)로 할 것인지 Pseudo 성분(Pseudo components)으로 할 것인지를 지정한다.
Reference Blend<3>에서
- Centroid of vertices 꼭지점의 중심으로 한다.
- Row ID: Workseet에 저장된 제약점을 사용한다.
- Coordinates : 동등한 값으로 분할한다.


☞ Pseudo성분

MINITAB에서 default로 성분의 하한을 0으로 지정한다. 그러나 어떤 성분은 반드시 들어가야 하기 때문에 하한을 0으로 지정해서는 안되며, 또한 어떤 성분은 일정량 이상 들어가서는 안 되는 경우도 있는데 이때 상한을 지정해야 한다. Pseudo 성분 데이터 영역을 rescale 하여 각 성분의 최소 허용 가능량을 0으로 한다.

예로, A와 B를 섞어서 되는 혼합물이 있을 경우 A+B 는 50이고, A는 반드시 20이상은 들어가야 한다. 그럴 경우 양, 비율, pseudo성분에 대한 실험 점은 다음과 같다





③ Curves...는 Plot에서의 선의 종류와 색깔을 지정한다.






④ Options...를 클릭



X축과 Y축의 척도의 최소, 최대를 설정할 수 있다.


⑤ OK를 클릭한다.


Graph 창에 나타난 결과




산화제, 연료 및 접착제의 양이 Reference Blend값 보다 작거나 클 때 탄성계수는 어떻게 변하는 가를 볼 수 있다. 위의 그림에서 알 수 있듯이 산화제의 경우 Reference Blend보다 클 경우에는 탄성계수가 일관되게 감소한다. 연료는 Reference Blend보다 클 경우에 탄성계수가 증가하다가 감소한다. 접착제는 Reference Blend보다 클 경우에 탄성계수가 감소함을 알 수 있다. Reference Blend는 각 성분의 양에 대한 기준이 될만한 값을 나타낸다.
