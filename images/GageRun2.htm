Gage Run Chart


1. 개요

현장에서 측정시스템을 분석하고 R&R값을 평가하면 종종 불만족스러운 결과도 나타나게 되는데, 이 때는 그 원인을 추적 조사하여 측정시스템을 개선해야만 한다. 이 경우, 부품별로 각 측정자의 측정값을 Run Chart로 도시하여 측정자의 측정 습관, 측정자와 부품간의 교호작용 등을 파악함으로써 문제의 원인을 체계적으로 조사할 수 있다. 이와 같이 Gage Run Chart는 Gage R&R결과 측정시스템이 만족스럽지 못할 경우, 원인을 추적하는데 사용할 수 있는데, 예를 통하여 그 사용방법을 알아 본다.

2. MINITAB을 사용한 Gage Run Chart (데이터 : Gage01.mtw)

L전자에서는 측정시스템을 분석하기 다음과 같이 데이터를 수집하였다. 10개 표본에 대하여 3명의 측정자가 2회 반복 측정하였다.

2.1 데이터의 입력

MINITAB Worksheet에 데이터를 입력한다

2.2 Gage Run Chart

① Stat > Quality Tools > Gage Run Chart

② Part number에는 Part, Operators에는 측정자 그리고 Measurement Data에는 측정값에 대한 데이터를 선택한다.

③ OK 버튼을 클릭한다.

작업자 2는 대부분의 경우 첫번째 측정치보다 두번째 측정치를 작게 측정하는 경향이 있으며 부품 10의 경우 각 작업자 별로 상이한 측정치를 보여 주고 있어 그 원인을 규명할 필요가 있음을 알 수 있다.
