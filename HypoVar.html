분산의 검정

 Stat> Basic Statistics > 2 Variances


MINITAB을 사용하여 둘 이상의 모집단에 대한 분산의 동일성 여부를 검정하고자 한다.



1.MINITAB을 사용한 분산의 검정 (데이터 : Hypo05.mtw )

어떤 화학약품의 제조에 상표가 다른 2종류의 원료를 사용하고 있다. 각 원료에서 주성분 A의 함량은 다음과 같다.



위의 데이터로부터 상표1과 상표2에 포함되어 있는 주성분 A의 함량의 분산이 동 일한 지를 유의수준 5%에서 검정하라.

상표1과 상표2의 분산을 각각 이라고 하면 분산의 동일성 검정에 대한 가설을 다음과 같이 설정할 수 있다.






1.1 데이터의 입력

MINITAB Worksheet에 아래의 데이터를 입력한다.




1.2 MINITAB의 실행

① Stat > Basic Statistics > 2 Variances 선택




② 데이터 열을 입력한다.






③ OK 버튼을 클릭한다.

[ Session 창에 나타난 결과 ]








1.3 결과 해석

두 가지 경우 모두에 대해 P-Value 가 0.05보다 크므로, 상표1과 상표2의 분산 이 다르다고 할 수 없다.

☞ 참고 : P-Value 란 무엇인가?

- P-Value는 귀무가설 H0를 기각했을 때, 우리의 판단이 잘못되었을 확률을 의미한다.
- 해석 : 만일 이 값이 작다면, 귀무가설을 기각한다.
