상자 그림 (Box Plot)

Graph>Box Plot



Box and Whisker Plot이라고도 하며 단일 그룹의 데이터 분포, 중심위치와 산포를 보여줄 뿐 아니라 여러 그룹간의 데이터 분포에 대한 차이점을 찾고 중심위치와 산포를 비교할 수 있다.


☞ Box Plot의 이해





1. MINITAB을 사용한 Box Plot의 작성 (데이터 : Graph03.mtw)

다음은 1999년 외국계 K기업에서 직원 92명을 대상으로 조사한 내용이다. 흡연여부에 따른 맥박수의 차이 및 중심위치, 산포를 파악하기 위하여 Box Plot을 작성하였다.




1.1 데이터의 입력

MINITAB Worksheet에 아래의 데이터를 입력한다




1.2 MINITAB의 실행

① Graph > Box Plot..을 선택한다.





② Graph1의 X, Y변수에 각각 "Smokes"와 "Pulse1"을 지정한다.





③ OK 버튼 클릭

[ Graph 창에 나타난 결과 ]




1.3 결과 해석

흡연자(smoke 1)의 맥박수가 비흡연자(smoke 2)에 비해서 산포가 넓게 나타나고 있어 흡연 자가 비흡연자에 비해 전체적으로 맥박수가 고르지 못하다는 것을 알 수 있다.
