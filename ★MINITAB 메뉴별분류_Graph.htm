MINITAB 메뉴별분류

2.Graph

대분류
설명
Plot
  두 변수 x, y 간의 상호관계를 나타내는 그래프. n개의 관측점의 위치를 xy의 2차 평면상에 찍어 놓은 도표. 여기서 두 변수 x, y 간의 관련성을 볼 수 있음
Chart
  여러 데이터에 따른 각 항목의 빈도 및 각 항목에 대한 여러 가지 통계량을 그래프가 주는 시각적인 효과로 비교 분석할 수 있게 함
Histogram
  자료를 계급별로 나누고 각 계급의 빈도인 돗수를 파악한 후, 가로축은 계급, 세로축은 돗수로 하여 막대그래프로 나타낸 그림으로 자료의 분포 형태와 중심위치 및 산포를 보여줌
Boxplot
 단일 그룹의 데이터 분포, 중심위치와 산포를 보여줄 뿐 아니라 여러 그룹간의 데이터 분포에 대한 차이점을 찾고 중심위치와 산포를 비교
Matrix plot
 두 개 이상의 변수들 간의 상관관계를 매트릭스 형태로 표시한 그래프. 즉, 여러 개의 산점도를 각 변수 별로 배열한 그래프
Dotplot
 데이터가 발생할 때마다 가로축에 수직으로 표시됨으로써 히스토그램처럼 프로세스의 중심 경향, 변동을 나타내며 데이터의 분포를 파악하기 용이
Pie chart
 데이터를 빈도에 따라 구분하여 전체에 대하여 특정한 변수의 비율을 시각적으로 보여주는 그래프
    Probability     Plot
 정규분포를 따르는 개별 데이터와 경험적 분포함수의 값을 타점하면 직선이 되도록 scale을 조정한 그래프
    Stem-and-    Leaf
 히스토그램에 비해 데이터 수가 비교적 적은 경우에 유용하며 원 데이터의 정보를 잃지 않고 분포 형태를 파악할 수 있음
