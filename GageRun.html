Gage Run Chart

Stat>Quality Tools>Gage Run Chart


Gage R&R 결과 측정시스템이 만족스럽지 못할 경우, 그 원인을 추적하는데 사용할 수 있다. 이하 예를 통하여 사용방법을 알아 본다.



1. 사용 방법 (데이터 : Gage01.mtw)

L전자에서는 측정시스템을 분석하기 다음과 같이 데이터를 수집하였다. 10개 표본에 대하여 3명의 측정자가 2회 반복 측정하였다.




1.1 데이터의 입력

MINITAB Worksheet에 데이터를 입력한다




1.2 MINITAB의 실행

① Stat > Quality Tools > Gage Run Chart




② Part number에는 Part, Operators에는 측정자 그리고 Measurement data에는 측정값에
    대한 데이터를 선택한다.




③ OK를 클릭한다.

[ Graph 창에 나타난 결과 ]




1.3 결과 해석

작업자 2는 대부분의 경우 첫번째 측정치보다 두번째 측정치를 작게 측정하는 경향이 있으며 부품 10의 경우 각 작업자 별로 상이한 측정치를 보여 주고 있어 그 원인을 규명할 필요가 있음을 알 수 있다.
