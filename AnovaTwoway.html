이원배치법(Two-way ANOVA)

Stat>ANOVA>Interactions Plot


두 개의 인자가 측정값에 어떠한 영향을 미치는가를 분석하는 방법이다. 두 개의 인자를 기준으로 각 수준별로 평균의 동일성과 교호작용의 효과를 조사하는 것이다.
교호작용(Interaction)이란 2개 이상의 인자 사이의 조합에서 발생하는 변동을 말한다. 예를 들어 인장강도에 영향을 미치는 변동요인으로 합성온도의 효과, 제조공법의 효과, 합성온도와 제조공법의 조합에서 이루어지는 교호작용의 효과, 실험오차 등으로 분해될 수 있을 것이다. 교호작용을 조사하려면 각 인자의 수준 조합에서 실험을 2회 이상 반복야 한다. MINITAB의 Stat > ANOVA > Interactions Plot를 이용하여 교호작용 Graph 창를 그릴 수 있다.
2개의 인자를 A와 B라 하고 A의 처리효과를 ai (i = 1, 2, · · ·, l), B의 처리효과를 bj (j = 1, 2, · · ·, m)라고 하면 이원배치법에서 검정하는 가설은 다음 세 가지이다.


a) 인자 A의 처리효과에 대한 가설

가설 : H0 : a1 = a2 = ... = a1 (인자 A의 각 수준에서의 처리효과가 같다.)
         H1 : H0가 아니다. (인자 A의 적어도 하나의 수준에서는 처리효과가 다르다.)

b) 인자 B의 처리효과에 대한 가설

가설 : H0 : b1 = b2 = ... = bm (인자 B의 각 수준에서의 처리효과가 같다.)
         H1 : H0가 아니다. (인자 B의 적어도 하나의 수준에서는 처리효과가 다르다.)

c) 인자 A와 B의 교호작용에 대한 가설

H0 : (ab)ij = 0 (i=1, 2, · · ·, l, j = 1, 2, · · · , m) (인자 A와 B 간의 교호작용이 유의하지 않다.)
H1 : H0가 아니다. (인자 A와 B 간의 교호작용이 유의하다.)

분산분석결과 유의수준이 0.05일 때 해당 항목에 대한 P-value가 0.05보다 작으면 귀무가설을 기각하고 해당 항목의 처리효과가 유의하다라는 결론을 내린다. 이원배치법을 적용하기 위해서는 데이터 (또는 잔차)의 정규성 및 등분산성, 독립성 등을 확인하여야 하며, 이는 과정에서 설명한다.


1. MINITAB을 사용한 이원배치법 (데이터 : Anova03.mtw)

특정 화학 제품을 압력관(Pressure vessel)에 통과시켜 생산하고 있다. 이 제품의 여과율(Filtration rate)에 대한 공정 요인들의 영향을 평가하기 위해 공정온도 2수준, 포름알데히드 농도 2수준에서 반복 4회씩 실험을 한 결과 여과율에 대한 데이터가 다음과 같다. 분산분석을 실시하려고 한다.





1.1 데이터의 입력

MINITAB worksheet에 아래의 데이터를 입력한다.





1.2 Main Effects Plot을 그려 본다.

분산분석에 앞서 다양한 형태의 시각적 분석도구를 통해 데이터를 분석함으로써 데이터의 특성 및 개선 포인트를 쉽게 잡아낼 수 있다.

① Stat > ANOVA > Main Effects Plot 선택





② Responses란에 '여과율'와 Factors란에 '온도' '농도'를 입력






③ OK 버튼을 클릭한다.

[ Graph 창에 나타난 결과 ]




공정온도 1수준과 포름알데히드 농도 1수준의 여과율이 높은 것을 볼 수 있다.



1.3 Interactions Plot을 그려 본다. (데이터 : Anova03.mtw)

① Stat > ANOVA > Interactions Plot 선택






② Responses란에 '여과율'을 입력하고 Factors란에 '온도', '농도'를 입력한다.






③ OK 버튼을 클릭

[ Graph 창에 나타난 결과 ]




위의 Graph 창에서 점들을 잇는 선이 평행하지 않으므로 온도와 농도간에는 교호 작용이 존재하는 것으로 보인다.


1.3 분산분석을 실시한다.

① Stat > ANOVA > Two-way 선택





② Response란에 '여과율', Row factor란에 '온도', Column factor란에 '농도'를 입력





③ OK 버튼을 클릭한다.

[ Session 창에 나타난 결과 ]



위 결과에서 온도의 P-value는 0.007이므로 온도는 여과율에 유의한 영향을 미치고 있으며, 농도의 P-value는 0.170이므로 농도의 효과는 유의하지 않고 온도와 농도간의 교호작용의 P-value는 0.018이므로 여과율에 유의한 영향을 미치고 있다고 볼 수 있다.
