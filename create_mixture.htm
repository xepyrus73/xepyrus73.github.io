혼합물 실험(Mixture Design)의 설계

Stat > DOE > Mixture > Create Mixture Design


1. 심플렉스 중심 배열법의 설계

어떤 특수한 폭약을 만드는데 들어가는 세 가지 성분(접착제, 산화제,연료)을 배합하여 탄성계수 (Modulus of elasticity)를 좋게 하여 주는 성분의 혼합비율을 찾으려고 한다. 단, 심플렉스 중심배열 을 이용하고 실험횟수는 10회로 한다.

① Stat > DOE > Mixture > Create Mixture Design…을 선택한다.



② Simplex centroid를 선택하고, Number of components 란에 3을 선택한다.




☞ 참 고
Display Available Designs...을 클릭을 하면 다음과 같은 표가 뜬다.



이 표를 통해, 성분 수와 그에 따른 혼합물 실험의 유형과 실험 수를 볼 수 있다. 예를 들어, 성분의 수가 3일 때 심플렉스 중심법인 경우의 실험 수는 7이다. 즉, 성분 수와 혼합물 실험의 유형에 따라 실험의 시행 횟수가 달라짐을 알 수 있다.


③ Designs...클릭한다.



심플렉스 중심법의 경우 성분수가 3일 경우 실험 수는 7이다. 여기에 실험의 축점(내점 또는 중심점)을 추가하기 위하여 Augment the design with axial points를 체크 한다. 축점을 추가하는 실험계획은 모델의 곡면적인 측면을 제공한다. 본 예에서는 축점을 추가하여 총 실험 수는 7 + 3(축점) = 10 이 된다.
Number of replicates for the whole design는 실험전체에 대한 실험 점의 반복 수를 지정하고, Number of replicates for the selected types of points은 선택된 실험 점의 반복 수를 지정한다. 본 예에서는 실험의 반복을 1로 지정한다.


④ Designs을 지정하고 나면 나머지 옵션 버튼들이 나타난다. 여기서 Components...를 클릭한다.

  • Total Mixture Amount에서 Single total 란에 혼합량의 합을 1로 지정 한다. 그러면 모든 성분의 양은 모든 성분의 양의 합이 1이 되게 비율적으로 계산된다. 또한 실제 값을 입력하면 실제 값에 따라 비율적으로 성분의 양이 계산된다.
  • Multiple totals 란은 혼합물의 총량에도 관심을 갖는 실험을 계획할 때 관심을 갖는 혼합물의 총량을 입력한다. 5개 까지 혼합물의 총량을 입력할 수 있다. 만약 알고 싶은 혼합물의 총량이 2개 있다고 가정하면 직접 2개의 총량을 입력하면 된다. 예를 들어 혼합물의 총량이 100인 경우와 200인 경우를 알고 싶다면 Multiple totals란에 100과 200을 입력하면 된다. 알고 싶은 혼합물의 총량이 2개인 경우 10회의 심플렉스 중심배열이 2개 생성되며 10*2=20회의 실험계획이 생성된다. .
  • Component Bounds Specified in Amount의 표에서는 성분명과 성분의 하한과 상한 값을 지정할 수 있다. 하한(Lower)의 경우 성분에서 반드시 들어가야 하는 양을 지정하고 상한(Upper)의 경우 들어갈 수 있는 최대의 양을 지정한다.

⑤ 실험의 계획 중 공정 변수들(Process Variables)이 있을 경우 선택한다.



공정변수는 성분의 혼합량 이외에 공정에서 고려되어지는 인자들이다. 예를 들어 반응 값에 혼합물(접착제, 산화제, 연료)의 성분비를 제외한 온도, 압력과 같은 인자 들이 영향을 미칠 수 있다. 이때 온도나 압력은 공정변수가 된다. 단, 공정변수의 경우 수준의 수는 2이며 7인자까지만 혼합물 실험계획에 추가하여 실험을 실시할 수 있다. 본 예에서는 공정변수가 없으므로 공정변수는 입력하지 않는다.


⑥ Options...에서는 실험을 랜덤하게 계획할 때 사용한다.



Base for random data generator 란은 랜덤하게 설계할 경우 초기값을 지정한다. 그러면 항상 동일한 랜덤순서를 얻을 수 있다. 초기값은 양의 정수를 입력해야 한다. Store design in worksheet를 클릭하면 랜덤하게 설계된 것을 Worksheet에 저장한다. Store design parameters in worksheet란을 클릭하면 실험계획과 관련된 값들(성분의 제약 조건, 선형 제약식,총량 등)을 워크시트에 저장한다.


⑦ Result...는 Session 창에 출력되는 형태를 선택한다.


  • Summary table은 설계된 실험에 대한 요약적인 내용을 출력해주며
  • Detailed description은 좀 더 상세하게 혼합량의 양, 실험공간의 복잡성, 실험 점의 유형, 성분의 한계값 등을 출력한다.
  • Data table은 완성된 실험계획을 보여준다.
본 예에서는 Detailed description and data table을 지정한다.


⑧ OK를 클릭한다.


Session 창에 나타난 결과



<1> 설계된 실험에 대한 요약(Summary table)으로서 성분(Component)은 3개이고 실험점 (Design point)은 10인 실험이 설계되었다.
<2> Point Type은 실험 점의 유형을 나타내는데 다음과 같다.
1 : 꼭지점(vertex point) 2 : 변점(edge point)
0 : 중심점(center point) -1 : 축점(axial point)
<3> Bounds of Mixture Components는 성분의 제약 조건(하한과 상한)과 관련한 사항들 이다.



완성된 실험계획으로 Run 8의 경우 접착제, 산화제,연료의 비율은 각각 0.1667, 0.1667, 0.6667로 실험 점은 축점(axial point)에 위치한다.


Worksheet에 나타난 결과 (Sim_Cen.mtw)



StdOrder는 Yate's의 표준배열 순서이고 RunOrder는 실험의 순서이다. 즉,Yates의 계산법에 의해 구해진 표준배열의 실험순서(StdOrder)를 랜덤하게 순서를 정해서 실행(RunOrder)한다. CenterPt가 1 이면 중심점이 없다(있다는 0)는 뜻이고 Blocks가 1 이면 블록화가 되지 않았다는 뜻이다.
중심점(Center point)을 추가 하여 실험을 하는 목적은 효과의 형태가 선형인지 비선형(곡선형태)인지를 알아 보기 위함이다. 물론, 중심점 추가에 따라 실험횟수가 증가된다. 인자의 수준이 계량적일 경우에는 쉽게 중심점을 추가할 수 있으나 인자의 수준이 정성적일 경우에는 중심점을 추가 시킬 수 없다.


2. 심플렉스 격자 배열의 설계

접착제를 만들기 위하여 A, B, C라는 3개의 성분이 들어간다. 가장 접착력이 좋은 제품을 만들기 위한 성분의 조합을 얻고자 (3, 3) 심플렉스 격자형 배열법을 하기로 결정하였다. 실험을 설계해보자. (단, Augmenting 설계를 하여 보다 좋은 결과를 얻도록 해보자)

① Stat > DOE > Mixture > Create Mixture Design..




② Simplex lattice를 선택하고, Number of components 란에 3을 선택한다.




③ Designs...을 클릭한다.


    본 예에서는 (3, 3)심플렉스 격자 배열법을 설계할 것이다.
  • Degree of lattice 란은 격자의 차수를 입력해야 하며, 이 예에서는 3을 선택한다. 실험의 중심점을 추가하기 위하여 Augment the design with center points를 체크 한다. 만약 실험에 축점(axial point)을 추가하려면 Augment the design with axial points를 체크 한다. 이러한 추가적인 설계로 생성된 점들은 모델의 곡면적인 측면을 제공한다.

  • Replicate Design Point에서 Number of replicates for the whole design는 실험전체에 대한 실험 점의 반복수를 지정하고, Number of replicates for the selected types of points은 선택된 실험 점의 반복수를 지정한다. 본 예에서는 실험의 반복을 1로 지정한다. 반복(replicate)은 동일한 처리에 대하여 실험을 2회 이상 실시하는 것이다. 반복의 횟수가 증가 함에 따라서 오차항 추정의 정밀도가 증가한다. 또한 반복횟수가 증가함에 따라서 구할 수 있는 교호작용 효과의 차수가 증가하게 된다.
④ Designs을 지정하고 나면 나머지 옵션 버튼들이 나타난다. 여기서 Components... 를 클릭한다.

  • Total Mixture Amount에서 Single total 란에 혼합량의 합을 1로 지정 한다. 그러면 모든 성분의 양은 모든 성분의 양의 합이 1이 되게 비율적으로 계산된다. 또한 실제 값을 입력하면 실제 값에 따라 비율적으로 성분의 양이 계산된다.

  • Multiple totals 란은 혼합물의 총량에도 관심을 갖는 실험을 계획할 때 관심을 갖는 혼합물의 총량을 입력한다. 5개 까지 혼합물의 총량을 입력할 수 있다. 만약 알고 싶은 혼합물의 총량이 2개 있다고 가정하면 직접 2개의 총량을 입력하면 된다. 예를 들어 혼합물의 총량이 100인 경우와 200인 경우를 알고 싶다면 Multiple totals 란에 100과 200을 입력하면 된다. 알고 싶은 혼합물의 총량이 2개인 경우 10회의 심플렉스 격자배열이 2개 생성되며 10*2=20회의 실험계획이 생성된다.

  • Component Bounds Specified in Amount의 표에서는 성분 명과 성분의 하한과 상한 값을 지정할 수 있다. 하한(Lower)의 경우 성분에서 반드시 들어가야 하는 양을 지정하고 상한(Upper)의 경우 초과해서 들어갈 수 없는 양을 지정한다.

⑤ 실험의 계획 중 공정 변수들(Process Variables)이 있을 경우 선택한다.



공정변수는 성분의 혼합량 이외에 공정에서 고려되어지는 인자들이다. 예를 들어 반응 값에 혼합물(접착제, 산화제, 연료)의 성분비를 제외한 온도, 압력과 같은 인자 들이 영향을 미칠 수 있다. 이때 온도나 압력은 공정변수가 된다. 단, 공정변수의 경우 수준의 수가 2이며 7인자까지만 혼합물 실험계획에 추가하여 실험을 실시할 수 있다. 본 예에서는 공정변수가 없으므로 공정변수는 입력하지 않는다.


⑥ Options...에서는 실험을 랜덤하게 계획할 때 사용한다.



Base for random data generator 란은 랜덤하게 설계할 경우 초기값을 지정한다. 그러면 항상 동일한 랜덤순서를 얻을 수 있다. 초기값은 양의 정수를 입력해야 한다. Store design in worksheet를 클릭하면 랜덤하게 설계된 것을 Worksheet에 저장한다. Store design parameters in worksheet란을 클릭하면 실험계획과 관련된 값들(성분의 제약 조건, 선형 제약식,총량 등)을 워크시트에 저장한다.


⑦ Result...는 Session 창에 출력되는 형태를 선택한다.


  • Summary table은 설계된 실험에 대한 요약적인 내용을 출력해주며
  • Detailed description은 좀 더 상세하게 혼합량의 양, 실험공간의 복잡성, 실험 점의 유형, 성분의 한계값 등을 출력한다. 본 예에서는 Detailed description and data table을 지정한다.
  • Data table은 완성된 실험계획을 보여준다.

⑧ OK를 클릭한다.

Session 창에 나타난 결과



설계된 실험에 대한 요약(Summary table)로서 성분(Component)는 3개이고 실험점(Design point)은 10인 실험이 설계되었다. 여기서 중심배열법과 다른점은 격자수(Lattice degree=3)이 나온다는 것이다.

Point Type은 실험 점의 유형을 나타내는데 다음과 같다.
1 : 꼭지점(vertex point) 2 : 변점(edge point)
0 : 중심점(center point) -1 : 축점(axial point)

Bounds of Mixture Components는 성분의 제약 조건(하한과 상한)과 관련한 사항들 이다.

Worksheet에 나타난 결과 (Sim_Latt.mtw)



StdOrder는 Yate's의 표준배열 순서이고 RunOrder는 실험의 순서이다. 즉, Yates의 계산법에 의해 구해진 표준배열의 실험순서(StdOrder)를 랜덤하게 순서를 정해서 실행(RunOrder)한다. CenterPt가 1 이면 중심점이 없다(있다는 0)는 뜻이고 Blocks가 1 이면 블록화가 되지 않았다는 뜻이다.
중심점(Center point)을 추가 하여 실험을 하는 목적은 효과의 형태가 선형인지 비선형(곡선형태)인지를 알아 보기 위함이다. 물론, 중심점 추가에 따라 실험횟수가 증가된다. 인자의 수준이 계량적일 경우에는 쉽게 중심점을 추가할 수 있으나 인자의 수준이 정성적일 경우에는 중심점을 추가 시킬 수 없다.

블록(block)은 실험의 반응값을 동일한 실험조건에서 얻기가 힘들 때 중요하지 않은 인자를 블록 설계를 통하여 교락 시키고자 할 때 사용된다.


3. 꼭지점 계획법의 설계

어떤 특수한 형태의 조명탄을 만들어 내기 위하여 다음과 같은 4개의 혼합성분이 사용되고 다음의 영역만이 실험상 흥미가 있는 영역이다.

0.4 ≤ 마그네슘 ≤ 0.6     0.1≤ 질산나트륨 ≤ 0.5
0.1 ≤ 질산스트론튬 ≤ 0.5     0.03 ≤ 접착제 ≤ 0.08

이 실험의 목적은 조명탄의 밝기가 인자의 어떤 조합에서 가장 강해지는가를 찾아 내고자 한다. 꼭지점 계획법을 실험을 계획하여라.

① Stat > DOE > Mixture > Create Mixture Design...을 선택한다.




② Extreme vertices를 선택하고, Number of components 란에 4을 선택한다.




③ Designs...을 클릭한다.


  • Number of replicates for the whole design는 실험전체에 대한 실험 점의 반복 수를 지정하고, Number of replicates for the selected types of points은 선택된 실험 점의 반복 수를 지정한다. 본 예에서는 실험의 반복을 1로 지정한다. 반복(replicate)은 동일한 처리에 대하여 실험을 2회 이상 실시하는 것이다. 반복의 횟수가 증가 함에 따라서 오차항 추정의 정밀도가 증가한다. 또한 반복횟수가 증가함에 따라서 구할 수 있는 교호작용 효과의 차수가 증가하게 된다.

  • 꼭지점 계획법(Extreme vertices design)에서 반응에 대한 모형을 2차 모형으로 생성된다고 가정하면 Degree of design차수를 2로 선택한다. 만일 2차식이 아니라 3차식이라면 3을 입력하면 된다. 추가적인 설계로 생성된 점들은 모델의 곡면적인 부분을 제공하는데 본 예에서는 추가(Augment)설계를 하지 않으므로 Augment the design with center points, Augment the design with axial points를 체크하지 않는다.

④ Designs을 OK하고 나면 나머지 옵션 버튼들이 나타난다. 여기서 Components... 를 클릭한다.


  • Total Mixture Amount에서 Single total 란에 혼합량의 합을 1로 지정 한다. 그러면 모든 성분의 양은 모든 성분의 양의 합이 1이 되게 비율적으로 계산된다. 또한 실제 값을 입력하면 실제 값에 따라 비율적으로 성분의 양이 계산된다.

  • Multiple totals 란은 혼합물의 총량에도 관심을 갖는 실험을 계획할 때 관심을 갖는 혼합물의 총량을 입력한다. 5개 까지 혼합물의 총량을 입력할 수 있다. 만약 알고 싶은 혼합물의 총량이 2개 있다고 가정하면 직접 2개의 총량을 입력하면 된다. 예를 들어 혼합물의 총량이 100인 경우와 200인 경우를 알고 싶다면 Multiple totals 란에 100과 200을 입력하면 된다. 알고 싶은 혼합물의 총량이 2개인 경우 10회의 심플렉스 격자배열이 2개 생성되며 10*2=20회의 실험계획이 생성된다.

  • Component Bounds Specified in Amount의 표에서는 성분 명과 성분의 하한과 상한 값을 지정할 수 있다. 하한(Lower)의 경우 성분에서 반드시 들어가야 하는 양을 지정하고 상한(Upper)의 경우 초과해서 들어갈 수 없는 양을 지정한다.

⑤ Linear Constraints..는 성분에 대한 제약식이 방정식으로 주어질 때 사용한다.



본 예에서는 성분에 대한 제약식이 방정식 형태가 아니고 상한과 하한으로 주어졌으므로 계수를 지정하지 않았다. 만일 0.5<2A+0.5B+C+0.2D<1 의 제약식이 있으면 Lower, A, B, C, D, Upper에 0.5, 2, 0.5, 1, 0.2, 1의 값을 각각 차례로 입력하면 된다.


⑥ 실험의 계획 중 공정 변수들(Process Variables)이 있을 경우 선택한다.



공정변수는 성분의 혼합량 이외에 공정에서 고려되어지는 인자들이다. 예를 들어 반응 값에 혼합물(접착제, 산화제, 연료)의 성분비를 제외한 온도, 압력과 같은 인자 들이 영향을 미칠 수 있다. 이때 온도나 압력은 공정변수가 된다. 단, 공정변수의 경우 수준의 수가 2이며 7인자까지만 혼합물 실험계획에 추가하여 실험을 실시할 수 있다. 본 예에서는 공정변수가 없기 때문에 공정변수를 입력하지 않는다.


⑦ Options...에서는 실험을 랜덤하게 계획할 때 사용한다.



Base for random data generator 란은 랜덤하게 설계할 경우 초기값을 지정한다. 그러면 항상 동일한 랜덤순서를 얻을 수 있다. 초기값은 양의 정수를 입력해야 한다. Store design in worksheet를 클릭하면 랜덤하게 설계된 것을 Worksheet에 저장한다. Store design parameters in worksheet란을 클릭하면 실험계획과 관련된 값들(성분의 제약 조건, 선형 제약식,총량 등)을 워크시트에 저장한다.


⑧ Result...는 Session 창에 출력되는 형태를 선택한다.



Summary table은 설계된 실험에 대한 요약적인 내용을 출력해주며 Detailed description은 좀 더 상세하게 혼합량의 양, 실험공간의 복잡성, 실험 점의 유형, 성분의 한계값 등을 출력한다. 본 예에서는 Detailed description and data table을 지정한다. Data table은 완성된 실험계획을 보여준다.

⑨ OK를 클릭한다.

Session 창에 나타난 결과



설계된 실험에 대한 요약(Summary table)로서 성분(Component)는 4개이고 실험점(Design point)은 20인 실험이 설계되었다. 여기서 반응이 2차식인 실험계획이 완성되었다. Point Type은 실험 점의 유형을 나타내는데 다음과 같다.

1 : 꼭지점(vertex point) 2 : 변점(edge point)
0 : 중심점(center point) -1 : 축점(axial point)

Bounds of Mixture Components부분에서 성분의 경계를 지정했기에 각 성분이 최대, 최소로 섞일 비율이 출력된다.


Worksheet에 나타난 결과 (extreme.mtw)



StdOrder는 Yate's의 표준배열 순서이고 RunOrder는 실험의 순서이다. 즉,Yates의 계산법에 의해 구해진 표준배열의 실험순서(StdOrder)를 랜덤하게 순서를 정해서 실행(RunOrder)한다. CenterPt가 1 이면 중심점이 없다(있다는 0)는 뜻이고 Blocks가 1 이면 블록화가 되지 않았다는 뜻이다.

중심점(Center point)을 추가 하여 실험을 하는 목적은 효과의 형태가 선형인지 비선형(곡선형태)인지를 알아 보기 위함이다. 물론, 중심점 추가에 따라 실험횟수가 증가된다. 인자의 수준이 계량적일 경우에는 쉽게 중심점을 추가할 수 있으나 인자의 수준이 정성적일 경우에는 중심점을 추가 시킬 수 없다.

블록(block)의 경우 실험의 반응값을 동일한 실험조건에서 얻기가 힘들 때 중요하지 않은 인자를 블록 설계를 통하여 교락 시키고자 할 때 사용된다.
