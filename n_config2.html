EASYTAB을 PC의 하드디스크에 저장하여 사용하면 PC가 네트웍에 연결되지 않은 때에도
언제든지 EASYTAB의 내용을 참조할 수 있습니다.

1. EASYTAB 초기화면에서 "다운로드"를 클릭하여 Easytab.zip 파일을 자신의 PC에
   저장합니다.
2. Easytab.zip 파일을 "밤톨이", "알집", "Winzip" 등의 압축 소프트웨어를 사용하여
   압축을 해제합니다. 이 때 압축 해제된 파일들은 "Easytab" 이라는 이름의 디렉토리로
   저장되도록 합니다.
3. Easytab 디렉토리 아래에 저장된 index_g.htm 파일을 클릭하면 EASYTAB을
   사용할 수 있습니다. index_g.htm 파일은 단축 아이콘을 만들어 Windows의 바탕
   화면으로 가져다 놓고 사용하는 것이 편리합니다.

☞ 단축 아이콘 만드는 방법
  • Windows 95의 경우
        Windows 탐색기에서 index_g.htm 파일을 선택한 후 마우스 오른쪽 버튼을 눌러
        단축 아이콘 만들기 를 선택하면 "index_g.htm의 단축 아이콘"이 생성됩니다.
        이 단축 아이콘을 Windows 바탕화면에 마우스로 드래그하여 가져다 놓습니다.
        (Drag & Drop)
  • Windows 98의 경우
        Windows 탐색기에서 index_g.htm 파일을 선택한 후 파일(F) 메뉴에서 보내기(T)
        - 바탕화면에 바로가기 만들기 를 선택하면 바탕화면에 단축 아이콘이 생성됩니다.
