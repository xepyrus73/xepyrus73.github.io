1. C 관리도의 개요

하나의 제품에 약간의 결점이 있어서 불량으로 처리되지 않고 사용 가능한 경우가 있다 예를 들면 전선, 직물, 두루마리 종이 제품 등의 경우, 일정한 길이 또는 일정 면적당 약간의 흠이 있어도 실제로 사용하는 데는 별 지장이 없을 수도 있을 것이나 그러한 결점의 수가 많아지면 제품의 기능이 저하되거나 가치가 감소될 것이다. 이와 같은 경우 단위 제품당 결점수를 관리하기 위해 고안된 관리도가 결점수 관리도이다. 결점수 관리도에서는 단위 제품당 결점수 X가 포아송 분포를 따른다고 가정한다. C 관리도는 검사단위 즉, 1회에 검사되는 양이 일정할 때 사용할 수 있는 결점수 관리도이다. 검사단위당 평균 결점수를 c라 할 때 C관리도의 중심선 및 관리 한계선은 다음과 같이 될 것이다.




만약 LCLc의 값이 음이 될 때는 LCLc=0으로 둔다. c 값을 모를 때에는 다음과 같이 예비 샘플로부터 c 값을 추정하여 사용한다.
= (발견된 총결점수)/(총검사 단위 제품수)
2. MINITAB을 사용한 C 관리도의 작성 (데이터 : C관리도.mtw)

어떤 망간 주조 제조 업체는 처리 전에 먼저 용접하는 새로운 형식의 주조방법이 결점수에 어떠한 영향을 미치는지에 관심을 갖고 있다. 과거의 경험에 의하면, 유사한 주조에 대한 결점수는 포아송(Poisson) 분포를 따른다. 하루의 작업시간동안에 단지 하나의 주조를 생산할 수 있고, 따라서 이용할 수 있는 자료는 한정되어 있다. 15개의 주조에 대하여 주조당 평균 7.9개의 결점이 관측되었다. 그러면 이것을 미래에 생산될 주조의 주조당 결점수의 추정값으로 사용할 수 있는가?
