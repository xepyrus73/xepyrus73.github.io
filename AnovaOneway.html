일원배치법(One-way ANOVA)

Stat>ANOVA>One-way


하나의 인자가 측정값에 어떤 영향을 미치는가를 조사하는 방법이다. 인자의 수준이 k개이면 각 수준을 적용한 측정값을 얻어서 수준 별 평균이 서로 동일한지 여부를 검정한다. 이것은 여러 모집단의 평균을 비교하는 것과 같다. 모집단 k의 평균을 μk라 하면 일원배치법에서 검정하는 가설은 다음과 같다.

가설 : H0 : μ1 = μ2 = ... = μk
          H1 : 적어도 하나의 평균이 다르다.

분산분석결과 유의수준이 0.05일 때 해당 항목에 대한 P-Value가 0.05보다 작으면 귀무가설을 기각하고 적어도 하나의 평균이 다르다는 결론을 내린다.
일원배치법을 적용하기 위해서는 데이터의 정규성 및 등분산성을 확인하여야 하며, 이를 검정하기 위해서는 MINITAB의 Normality Test 와 Test for Equal variances를 활용한다.



1. MINITAB을 사용한 일원배치법 (Anova01.mtw)

같은 등급의 서로 다른 5가지 소형 자동차(A, B, C, D, E)의 연비(리터당 평균 주행거리)를 비교하기 위해 소형 자동차들의 연비를 측정한 결과 다음 데이터를 얻었다. 자동차별로 연비에 차이가 있다고 할 수 있는가 분석하라.



가설 : H0 : μA = μB = μC = μD = μE
         H1 : 적어도 하나의 평균이 다르다.

1.5 결과 해석

[ Session 창에 나타난 결과 ]


DF는 자유도(Degree of Freedom)이고 자동차의 자유도는 4, 오차항(Error)의 자유도는 18이며 총 자유도는 자동차와 오차항의 자유도를 더한 4+18=22가 된다.
SS는 제곱합(Sum of Square)이며 이는 측정값의 변동을 나타내는 값이다. 자동차의 변동은 27.323, 오차항의 변동은 17.284이며 총변동은 27.323+17.284=44.606이 된다.
MS(Mean Square)는 평균제곱이며 이는 제곱합을 자유도로 나눈 값이다. 따라서 자동차의 평균제곱은 27.323/4=6.831이고 오차항의 평균제곱은 17.284/18=0.960이 된다.
F는 F-통계량이며 자동차의 평균제곱을 오차항의 자유도로 나눈 값이다. 여기서 F-통계량은 6.831/0.960=7.11이 된다. 이 값은 자유도가 자동차의 자유도 4, 오차항의 자유도 18인 F 분포를 따르며 P-Value를 계산하는데 사용된다. 우리는 이 예제에서 0.014의 P-Value를 얻었다.
