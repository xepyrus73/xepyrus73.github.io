등분산성의 검정(Test for Equal Variances)

Stat>ANOVA>Test for Equal Variances


분산분석(Analysis of Variance)을 포함한 여러 통계 분석 방법들은 비교하고자 하는 서로 다른 표본들의 모집단은 같은 분산을 가지고 있다는 가정을 가지고 있다. 모집단간의 분산이 같은 지의 여부를 검정하기 위해 MINITAB의 Test for Equal Variances…를 사용한다.

1.MINITAB을 사용한 등분산성의 검정 (데이터 : Anova01.mtw)

같은 등급의 서로 다른 5가지 소형 자동차(A, B, C, D, E)의 연비(리터당 평균 주행거리)를 비교하기 위해 소형 자동차들의 연비를 측정한 결과 다음 데이터를 얻었다. 자동차별로 연비에 차이가 있다고 할 수 있는가 분석하기 전에 5가지 자동차간에 분산의 차이가 있는지를 검정하고자 한다.



가설 : H0 :
        H1 : 적어도 하나의 분산이 다르다.


1.1 데이터의 입력
    Worksheet에 다음과 같이 데이터를 입력한다.




① Stat > ANOVA > Test for Equal Variances...을 선택한다.




② Response란에는 '연비'를 Factors란에는 '자동차'를 선택한다.



Response는 반응값이 있는 열을 지정한다. 본 예에서는 ' C2 연비' 이다.
Factors는 인자가 있는 열을 지정한다. 본 예에서는 ' C1 자동차' 이다.
Confidence level란은 신뢰수준을 얼마로 할 것인지를 지정한다. 여기서는 신뢰수준을 95%로 하기 위하여 95.0을 입력한다.


③ Stroage...를 선택하여 Worksheet에 저장하고자 하는 값을 지정한다.



차례로 표준편차(Standard deviation), 분산(Variance), 신뢰구간의 상한값과 하한값이다.


④ OK를 클릭한다.

Worksheet에 나타난 결과



C3 열부터 각각 표준편차(STDV1), 분산(VARS1), 신뢰구간의 상한값(UPPE1)과 하한값 (LOWE1)을 나타낸다. 그리고 1행부터 차례로 자동차의 인자의 수준(A,B,C,D,E)별로 나온다. 예를 들어 자동차 C의 연비에 대한 표준편차는 1.22678, 분산은1.505, 95% 신뢰구간은 (0.636480 5.39293)이다.

Session 창에 나타난 결과



<1> 인자의 수준별로 표준편차(Sigma)와 신뢰구간의 하한(Lower)과 상한(Upper)이 나와 있다. 각 인자의 관측수(N)도 알 수 있다.
<2> 연비의 데이터가 정규분포 데이터인 경우 Bartlett's Test의 P-value를 확인한다.
<3> 연비의 데이터가 연속형 데이터인 경우 Levene's Test의 P-value를 확인한다.

Graph 창에 나타난 결과


  • Bartlett's Test
    데이터가 정규분포로부터 나왔을 때 수행되는 것으로 만약 인자의 수준이 2수준이라면 F-Test가 수행된다.
  • Levene's Test
    데이터가 어떤 분포인지는 알 수 없어도 연속형일 경우에 수행된다.
Session 창과 Graph 창의 결과를 보면 Bartlett's Test와 Levene's Test에서 P-Value가 각각 0.794와 0.799로 큰 값을 가진다. 특히 Graph 창에서는 신뢰구간이 비 대칭적으로 나오는데 이는 데이터가 카이제곱(Chi-Square) 분포를 따름을 알 수 있다. 유의수준을 0.05로 할 경우 분산이 같다는 귀무가설을 기각할 수 없으므로 분산이 다르다고 할 수 없다.
