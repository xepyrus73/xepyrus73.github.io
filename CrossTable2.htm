Cross Tabulation

Stat>Tables>Cross Tabulations


원시 데이터(Raw data)를 분할표(돗수데이터)로 변환

MINITAB의 Stat>Tables>Cross Tabulations를 이용하여 원시 데이터를 카이제곱 검정에서 사용하는 rxc 분할표(Contingency Table)로 손쉽게 정리할 수 있다. 계산결과는 Session 창에 출력된다.

1. MINITAB 사용 방법 (데이터 : Hypo07.mtw)

어느 공장에서 A, B, C 3 대의 기계로 생산된 제품을 기계별로 각각 100개씩 샘플을 뽑아 양/불량을 검사한 결과를 담은 원시 데이터(Raw data)를 분할표로 정리한 결과가 다음 표와 같다. 원래의 데이터로부터 MINITAB의 Stat>Tables>Cross Tabulations를 이용하여 rxc 분할표를 작성하라.



1.1 데이터의 입력
    MINITAB Worksheet에 아래의 데이터를 입력한다.




1.2 MINITAB의 실행

① Stat>Tables>Cross Tabulations 선택




② Classification variables 란에 '기계'와 '판정결과'를 입력한다. Display의 Counts를 체크한다.




[Summaries] 설명

☞ Cross Tabulations -> Summaries -> Proportion equal to:
분할표의 각 셀 중 특정 값의 비율을 갖는 셀을 출력하고자 할 때 선택한다. 우측 빈 칸에 원하는 비율을 입력한다.

☞ Cross Tabulations -> Summaries -> Proportion between ~ and ~:
분할표의 각 셀 중 특정 범위의 비율을 갖는 셀을 출력하고자 할 때 선택한다. 우측 빈 칸 중 첫 번째에 비율의 하한, 우측에 상한을 기입한다.


③ OK 버튼을 클릭한다.

Session 창에 나타난 결과



원시 데이터로부터 카이제곱 검정에서 사용하는 데이터 구조인 3´2 분할표를 생성했다.



Cross Tabulation을 사용한 동일성 검정 (데이터 : Hypo07.mtw)

어느 공장에서 A, B, C 3 대의 기계로 생산된 제품을 기계별로 각각 100개씩 샘플을 뽑아 양/불량을 검사한 결과를 담은 현장 데이터를 분할표로 정리한 표이다. 이 데이터로부터 직접 기계별로 불량률이 서로 다르다고 할 수 있는지 a=0.05로 검정하라.



귀무가설 H0 : 기계 A, B, C의 불량률은 모두 동일하다.
대립가설 H1 : H0가 아니다.


1.1 데이터의 입력
    MINITAB Worksheet에 아래의 데이터를 입력한다.




1.2 MINITAB의 실행

① Stat > Tables > Cross Tabulations 선택




② Classification variables 란에 '기계'와 '판정결과'를 입력한다. Chi-Square analysis를 체크
    하고 돗수와 기대돗수를 표시하기 위해 Above and expected count를 체크한다.



☞ Cross Tabulations -> Chi-Square analysis
다음과 같은 출력물을 표시하고자 할 때 선택한다.
Show count: 각 속성의 관측도수를 표시한다.
Above and expected count: 각 속성의 관측도수와 기대도수를 표시한다.
Above and std.residual: 각 속성의 관측도수, 기대도수, 그리고 표준잔차를 표시한다.

☞ Cross Tabulations -> Frequencies are in
워크시트에 입력된 데이터가 원시 데이터(Raw data)가 아닌 돗수 데이터일 때 선택한다. 물론 돗수 데이터를 포함하고 있는 열을 지정한다.

☞ Cross Tabulations -> Options
다음과 같은 Options 메뉴가 있다.




③ OK 버튼을 클릭한다.

Session 창에 나타난 결과



관측돗수와 귀무가설 하에서 기대돗수가 계산되었으며 카이제곱 통계량은 3.116이다. 그리고 P-Value가 0.211이므로 H0를 기각할 수 없다. 즉, 기계별로 불량률이 다르다고 할 수 없다.
