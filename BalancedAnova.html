Balanced ANOVA

Stat>ANOVA>Balanced ANOVA


Balanced ANOVA는 다수의 인자에 대하여 갼 주효과 및 교호작용 등을 분석한다. 모수모형, 혼합모형 등을 분석할 수 있으며 적절한 구조식을 입력하여 여러 형태의 실험모형을 분석할 수 있다. 또 각 처리 수준의 조합에서 얻어진 측정값의 개수가 모두 같은 데이터인 균형설계(Balanced Design) 데이터를 분석한다. 불균형설계 (Unbalanced Design) 데이터는 General Linear Model에서 분석한다.

데이터 구조의 MINITAB 표현 : 인자가 A, B, C 세 개일 경우 A의 효과, B의 효과, C의 효과, A와 B의 교호작용, A와 B와 C의 교호작용을 분석하려면 MINITAB의 대화상자의 Model란에 'A B C A*B A*B*C '로 입력한다. 만일 A의 효과, B의 효과, C의 효과, A와 B의 교호작용 만을 분석하려면 Model란에 'A B C A*B'를 입력한다.

Balanced ANOVA를 적용하기 위해서는 데이터의 정규성 및 등분산성을 확인하여야 하며, 이를 검정하기 위해서는 MINITAB의 Normality Test 와 Test for Equal variances를 활용한다. Test for Equal variances는 과정에서 설명한다.
