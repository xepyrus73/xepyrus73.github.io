기술통계량(Descriptive Statistics)의 산출

Calc> Column Statistics


1. 사용 방법 (데이터 : DStat01.mtw)

두 개의 주사기로 화학 약품을 병 속에 주입하는 공정에서 주사기별로 주입량의 평균과 산포가 얼마나 되는지 확인하기 위해 각 주사기별로 약품이 충전된 병을 5개씩 샘플링하여 데이터를 수집하였다.


1.1 데이터의 입력

하나의 모집단으로부터 얻어진 데이터가 한 조만 있을 경우에는 데이터를 하나의 Column에 입력하면 되지만, 주어진 조건과 같이 두 모집단으로부터 추출된 데이 터가 두 조 이상 있을 경우는 별개의 Column에 데이터를 입력할 수도 있고, 하 나의 Column에 입력하되 데이터를 구별해주는 Index Column을 함께 입력할 수 도 있다. 또, 별개의 Column에 입력한 후 Stack기능을 이용하여 두 Column을 합칠 수도 있다.





1.2 MINITAB의 실행

① Calc> Column Statistics 선택





② 구하고자 하는 기술통계량을 지정하고 Input variable 란에는 데이터가 들어 있는
    Worksheet의 Column을 지정한다.





③ OK를 클릭한다.
