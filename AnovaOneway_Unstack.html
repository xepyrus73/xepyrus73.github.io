One-Way ANOVA (Unstacked)

Stat>ANOVA>One-way (unstacked)


1. 사용 방법 (데이터 : Anova02.mtw)

이번에는 데이터가 Unstacked 형태인 경우 일원배치 분산분석을 수행한다. PVC Resin 입자의 평균 입경은 중합공정의 교반기 종류의 영향을 받는 것으로 예상되어 교반기 종류 4수준(A, B, C, D)에 대하여 5회씩 실험을 실시하였다. 교반기 종류별로 유의한 차이가 있는지 조사하라.





1.1 데이터의 입력

MINITAB worksheet에 아래의 데이터를 입력한다.





1.2 Interval Plot을 그려본다.

Interval Plot를 그리려면 먼저 데이터를 Stack해야 한다.

① Manip > Stack > Stack Columns 선택




② Stack the following columns란에 A B C D를 입력하고 Column of current Worksheet 란에
    '입경'을 입력하고 Store subscripts in란에 '종류'를 입력






③ OK 버튼을 클릭

[ Worksheet에 나타난 결과 ]




A열에서 D열까지 데이터가 Stack되어 '입경'과 '종류'에 저장되었다.



④ Stat > ANOVA > Interval Plot 선택




⑤ Y variable 란에 '입경'와 Group variable란에 '종류'를 입력






⑥ OK 버튼을 클릭

   [ Graph 창에 나타난 결과 ]




교반기 A와 B의 입경이 교반기C와 D에 비해 작으며 표준오차 구간도 넓어 보인다.



1.3 분산분석을 실시한다.

① Stat > ANOVA > One-way (Unstacked) 선택





② Responses란에 A-D를 입력한다. 이것은 A B C D를 입력한 것과 같다.




③ OK 버튼을 클릭한다.

[ Session 창에 나타난 결과 ]



위의 결과에서 P-value가 0.101이므로 교반기 종류가 평균입경에 유의한 영향을 미친다고 볼 수 없다.
