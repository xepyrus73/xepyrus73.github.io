한글 검색어를 사용한 검색이 안될 때 웹 브라우저의 환경 설정
변경 방법 (영문 NETSCAPE Communicator V4.7)



1. Netscape의 Edit 메뉴에서 Preferences… 선택




2. Category중 Appearance의 Fonts를 누르고 "For the Encoding :" 에서 Korean을
    선택한다



3. Category중 Navigator의 Languages를 누르고 오른쪽 창에 "Korean [ko]"가 표시되어
    있는 지를 확인한다. "Korean[ko]" 외의 기타 언어는 선택 후 창의 오른쪽에 있는
    Delete 버튼을 눌러 제거한다.




4. 만약 "Korean[ko]" 가 없으면 Language창의 오른쪽에 있는 Add… 버튼을 눌러 아래의
    "Add Languages" 창에서 "Korean[ko]"를 선택하고 OK 버튼을 누른다.




5. Preference 창의 OK 버튼을 누른다.
