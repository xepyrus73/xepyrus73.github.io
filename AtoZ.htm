A ~ Z

Bartlett's Test
등분산성을 검정하는 방법으로 데이터가 정규분포로부터 나왔을 때 수행되는 것으로 만약 인자의 수준이 2수준이라면 F-Test가 수행된다.

☞ 참고 : 분산의 검정


Box-Cox 변환(Box-Cox Transformation)
데이터가 정규분포를 따르지 않을 때, 특히 한쪽 꼬리가 긴 치우친 형태의 분포를 따르는 데이터를 정규분포로 변환하고자 할 때 사용할 수 있는 도구이다. Box-Cox 변환은 원 데이터를 X, 변환된 데이터를 Y로 나타낼 때, 다음과 같은 식을 사용하여 변환한다.



☞ 참고 : Box Cox 변환

C 관리도(C Chart)
단위 제품당 결점수를 관리하기 위해 고안된 관리도.
Chart : 여러 데이터에 따른 각 항목의 빈도 및 각 항목에 대한 여러 가지 통계량을 그래프가 주는 시각적인 효과로 비교, 분석할 수 있게 한다.

☞ 참고 : C 관리도


Coded
분석시 인자수준의 값이 낮은수준은 -1, 높은수준은 +1이 되도록 각 인자의 수준을 재조한 것이다.


Contour Plot
이차원의 평면 그래프로 이 Plot은 만족할 만한 반응값을 확인하고 조건을 잡는데 유용하다. 두 인자들이 반응변수에 어떻게 영향을 주는지에 대해 분석할 수 있다.

☞ 참고 : 요인배치 실험의 Contour Plot


Crossed
A와 B 인자가 있을 때 인자 A의 각각에 수준에 대해 인자 B의 각각의 수준이 대응되어 실험이 일어나는 것이다.

Durbin-Waston Statistics
회귀분석에서 잔차들의 자기상관 정도를 파악하기 위해 사용하는 통계량.


Effects plots
반응치에 영향을 미치는 인자를 확인하기 위하여 인자의 효과의 상대적인 크기를 비교하고 통계적인 유의성을 검정해 준다.


Family error rate
다중비교에서 수준별 평균들간의 실제 차이를 포함하지 않는 신뢰구간이 하나이상 얻어질 최대 확률이다.


Fold
기존의 실험계획에서 각 요인에 대한 반대 부호를 사용하여 교락된 일부실시의 실험계획에서 교락을 줄이기 위해 사용하는 방법
(예 : Orginal Fraction Folded on all Factors )




Gage R&R
계측기의 능력을 평가하는 데는 일반적으로 반복성과 재현성을 동시에 고려한 척도를 이용하는데, 이것이 바로 R&R이다.

☞ 참고 : Gage R&R


I-MR 관리도(I-MR Chart)
개별 측정치 X에 의하여 공정 평균과 산포를 동시에 관리하기 위한 관리도. 공정 평균을 관리하기 위해서는 개별 측정치 X가 이용되고, 산포를 관리하기 위해서는 인접한 두 측정치의 범위인 이동범위 MR을 이용한다.

☞ 참고 : I-MR 관리도


I-MR-R/S 관리도 (I-MR-R/S Chart)
Individual 관리도, MR관리도, R관리도 혹은 S관리도를 하나의 화면에 나타내 준다. 여기서 먼저 Individual 관리도는 공정의 평균이 관리 상태인지를 판단하고, MR관리도는 부분군 간의 변동이 관리 상태인지를 판단한다. 또한 R관리도 혹은 S관리도는 부분군내의 변동이 관리 상태인지를 판단해 준다.

☞ 참고 : I-MR-R/S 관리도


Individual 관리도(Individual Chart)
공정평균을 관리함에 개별 측정치 X를 사용하는 관리도이다. 시간, 비용 또는 공정 특성상의 이유로 여러 개의 제품으로 하나의 부분군을 형성하기 어려운 경우에 사용된다.

☞ 참고 : Individual 관리도


Individual error rate
다중비교에서 주어진 신뢰구간이 그룹 평균내의 실제 차이를 포함하지 않을 확률이다.


Interval Plot
인자의 각 수준 별로 특성치의 평균 값, 표준오차 또는 신뢰구간을 도시해 준다.


IQR(Interquartile Range)
3/4분위수와 1/4분위수의 차, 즉, Q3 - Q1로 계산되며 이상치가 있을 경우 범위보다 나은 산포의 척도가 될 수 있다.


Levene's Test
등분산성을 검정하는 방법으로 데이터가 어떤 분포인지는 알 수 없어도 연속형일 경우에 수행된다.


Matrix Plot
두 개 이상의 변수들 간의 상관관계를 표시한 그래프이다. 즉 여러 개의 산점도를 각 변수 별로 배열한 것이다. 한 개의 그래프로 여러 변수에 대한 관련성을 동시에 파악할 수 있다.

☞ 참고 : Matrix Plot


MSSD(Mean of Successive Squared Difference)
이웃하는 데이터간 차이의 제곱을 평균하여 2로 나눈 값. 예로서 데이터가1, 3, 2, 1라면 MSSD는 을 2로 나눈 값으로 1이 된다.


Multi-vari Chart
다수의 요인들이 특성치에 영향을 미칠 때, 각 요인 별로 특성치에 미치는 효과를 표시한 그래프이다. 측정값의 요인 별 경향과 관련성을 시각적으로 파악할 수 있다.

☞ 참고 : Multi-vari chart


Nested
인자 A의 하나의 수준내에 B의 각각의 수준이 대응되어 실험이 일어나는 것이다.


Normal Probability Plot
반응치에 영향을 미치는 치명적이지만 소수인자 또는 중요변수를 확인하는데 사용한다. 효과의 상대적인 크기를 비교한다.


np 관리도(np Chart)
불량개수 관리도라고도 하며 부분군의 크기가 일정한 경우 불량률을 관리하는 p 관리도 대신 계산상의 편의를 고려하여 부분군에 포함된 불량개수를 관리하기 위한 관리도.

☞ 참고 : np 관리도


Paired t 검정
데이터가 동질적인 쌍의 형태를 띨 때, 두 집단의 평균치 비교를 위해 활용되는 MINITAB 분석 메뉴이다.


P 관리도(P Chart)
불량률 관리도라고도 하며 부분군의 불량률을 이용하여 공정의 불량률을 관리하기 위한 관리도.

☞ 참고 : p 관리도


Pie Chart
데이터를 빈도에 따라 구분하여 전체에 대하여 특정한 변수의 비율을 시각적으로 보여주는 그래프이다.


Plackette-Burman
분해능이 III인 실험 계획으로 주효과를 검출하는데 사용한다. 주효과는 2차 교호작용과 별명 관계이므로 2차 교호작용은 무시해도 좋다는 가정이 있어야 한다. 그러면 주효과는 정도 좋게 검출 된다. MINITAB에서 이 방법을 지원해 준다.

Pseudo성분
어떤 성분이 일정량이상 들어가야 하거나 들어 가서는 안 되는 경우의 실험을 제약된 실험이라 하는데 이 실험은 매우 높게 상관관계된 계수를 생산한다. 이를 줄이기 어떤 성분의 최소허용 가능량을 0 으로 재 조정한 성분을 Pesudo 성분이라 한다.


P-Value
검정통계량에 의하여 귀무가설을 기각하게 되는 유의수준의 최소값, 즉, 검정통계량의 관측값에 의해 귀무가설을 기각할 경우 부담하게 될 위험의 크기를 나타낸다. 가설검정에서 귀무가설 (H0)이 거짓이라는 결론을 내릴 때, 이 결론이 잘못되었을 확률을 말한다. 따라서 P-value가 유의수준 a보다 작으면 귀무가설을 기각하고 대립가설을 채택한다.


R 관리도 (R chart)
각 부분군의 범위를 관리도에 타점하여 공정의 산포를 관리하기 위한 관리도.

☞ 참고 : R 관리도


Reference blend
각 성분의 양에 대한 기준이 될만한 값을 나타낸다.


Response Trace Plot
각각의 성분이 어떻게 반응에 영향을 미치는지를 reference blend와 상대적으로 비교한다.
각각의 성분들이 반응변수에 어떻게 영향을 주는지에 대해 분석할 수 있다.


S 관리도(S Chart)
부분군의 표준편차를 관리도에 타점하여 공정의 산포를 관리하기 위한 관리도로 부분군의 크기가 큰 경우에는 R 관리도보다 s 관리도를 적용하는 것이 더 효율적이다.

☞ 참고 : S 관리도


Simplex Design Plot
설계된 실험계획을 시각적으로 이해하기 위하여 MINTAB에서는 삼각좌표를 이용하여 실험점과 실험영역을 표시해준다. 이러한 삼각좌표의 실험 점들은 실험에서 사용하고 있는 조합으로서 혼합물 실험의 경우 3개의 성분에 대한 실험계획을 보여준다.


Surface (wireframe) Plot
3차원의 표면으로 출력되는 그래프로 이 Plot은 만족할 만한 반응값을 확인하고 조건을 잡는데 유용하다.


Symmetry plot
표본 데이터가 좌우 대칭인 모집단 분포로부터 추출된 것인지 여부를 알고자 할 때 사용되는 그래프로서, 데이터가 완벽한 좌우 대칭이라면 모든 점들이 원점을 지나는 기울기 1인 직선 상에 타점된다.


U 관리도(U Chart)
단위당 결점수 관리도라고도 하며 결점수 관리도에서 공정변화의 탐지능력(민감도)이나 경제적 요인들을 고려하여 여러 개의 검사단위를 묶어서 하나의 부분군을 형성할 수 있다. 이러한 경우에 공정을 단위 크기 당 결점수에 의하여 관리하기 위한 관리도.

☞ 참고 : U 관리도


Uncoded
실험계획의 배치시 인자 수준을 "+, -" 또는 "1, 0"이 아닌 실제의 값으로 입력한 값이다.


V 마스크(V-mask)
누적합 관리도에서 관리상태 이탈 여부를 판정하는데 사용되는 V자형 도형. V 마스크는 다음의 그림에서 점 o와 p사이의 거리인 d와 v자의 각도 θ에 의하여 결정된다. V마스크를 이용한 공정에 대한 판정은 아래 그림에서와 같이 V 마스크의 수평이 유지된 상태에서 가장 최근의 부분군의 자료에 해당되는 점을 p와 일치시킨 후에 관리도 상의 모든 점들이 V마스크 안쪽에 놓이면 공정이 관리상태에 있다고 판정하고, 한 점이라도 마스크에 가리워 지면 관리 일탈 상태로 판정한다.



Xbar 관리도(Xbar Chart)
각 부분군의 평균을 관리도에 타점하여 공정평균을 관리하기 위한 관리도. 다른 관리도에 비하여 공정의 갑작스러운 변화를 빨리 탐지하는 장점을 가지고 있다.

☞ 참고 : X bar 관리도


Xbar-R 관리도(Xbar-R Chart)
부분군의 평균과 범위를 이용하여 공정평균과 산포를 동시에 관리하기 위한 관리도. Xbar 관리도와 R 관리도를 함께 사용하는 방법이다.

☞ 참고 : X bar-R 관리도


Xbar-s 관리도(Xbar-s Chart)
Xbar-R 관리도에서 부분군의 크기가 비교적 클 때는(n>6) 범위 R을 이용하는 것 보다 표준편차 s를 이용하는 것이 더 효율적이다. Xbar-R 관리도에서 R관리도 대신 s관리도를 사용하는 경우를 Xbar-s 관리도라고 한다.

☞ 참고 : X bar-S 관리도


Z-MR 관리도 (Z-MR Chart)
표준화된 개별 관측치(Z)와 short run 공정으로부터 이동범위 (Moving Range, MR)를 그리는 관리도로 단기 프로세스에서 각각의 Run에 대해 충분한 데이터를 얻지 못할 때 사용한다. Z-MR 관리도에서는 측정치 데이터를 표준화하는데, 표준화는 각각의 다른 Run으로부터 수집된 데이터를 하나의 관리도를 통해 해석이 가능하게끔 한다.

☞ 참고 : Z-MR 관리도


1/4 분위수(Q1)
데이터를 작은 값부터 크기 순으로 나열했을 때, 번째에 해당하는 수

3/4 분위수(Q3)
데이터를 작은 값부터 크기 순으로 나열했을 때, 번째에 해당하는 수
