1-Sample t 검정

Stat> Basic Statistics> 1 Sample t


일반적으로 표준편차를 정확히 아는 경우는 매우 드물다. 따라서 1-Sample Z 검정의 적용 범위는 상당히 제한적이다. 표준편차를 모를 때, 모평균의 검정을 위해서는 1-Sample t 검정을 적용한다. 1-Sample Z 검정과 마찬가지로 데이터의 정규성 여부를 검토하여야 하며, 이는 MINITAB의 Normality Test를 통해 확인할 수 있다. 단, 시료수가 많은 경우 (보통 30개 이상), 정규성을 확인할 필요가 없다.


1. MINITAB을 사용한 t 검정 (데이터 : Hypo02.mtw )

엔진블록과 실린더 헤드 커버간의 간격이 크면 오일의 누수(leak)가 발생하게 된다. 이 간격의 규격은 0.8mm 이하여야 한다. 따라서 30개의 데이터를 랜덤하게 수집하여 간격이 0.8mm 이하인지를 유의수준 5%에서 검정하여라.


검정 대상값이 0.8mm이고 수율이 0.8mm이하인지를 검정하는 것이므로 다음과 같이 가설을 설정한다.






1.1 데이터의 입력

MINITAB Worksheet에 아래의 데이터를 입력한다




1.2 MINITAB의 실행

① Stat > Basic Statistics > 1-Sample t 선택





② Variables란에 간격의 데이터열을 입력하고 Test mean란에는 검정 대상값 μ0 를 입력한다







③ OK 버튼 클릭한다.

[ Session 창에 나타난 결과 ]
1.3 결과 해석

P-Value 가 0.027<0.05 이므로 H0를 기각한다. 즉, 간격의 규격이 0.8mm 이하 라고 말할 수 있다.
