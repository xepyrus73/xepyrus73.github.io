Basic Statistics
  Display Descriptive  Statistics   다양한 기술통계량의 값과 그래프들을 화면 출력한다.
 Store Descriptive  Statistics   기술통계량의 값을 계산하여 Worksheet에 저장한다.
 1 Sample Z   모집단의 표준편차를 알 때, 한 개 모집단의 평균치에 대한 신뢰구간 계산과 가설검정을 실시한다.
 1 Sample t   모집단의 표준편차를 모를 때, 한 개 모집단의 평균치에 대한 신뢰구간 계산과 가설검정을 실시한다.
 2 Sample t   모집단의 표준편차를 모를 때, 두 모집단의 평균치 차에 대한 신뢰구간 계산과 가설검정을 실시한다.
 Paired t   데이터가 동질적인 쌍의 형태를 띄는 두 집단의 평균치 차에 대한 신뢰구간 계산과 가설검정을 실시한다.
 1 Proportion   한 개 모비율에 대한 신뢰구간 계산과 가설검정을 실시한다.
 2 Proportions   두 모집단의 모비율 차에 대한 신뢰구간 계산과 가설검정을 실시한다.
 2 Variances   두 모집단의 등분산성 여부에 대한 가설검정을 실시한다.
 Correlation   두개의 변수들간의 상관계수를 계산한다.
 Covariance   두개의 변수들간의 공분산을 계산한다.
 Normality Test   연속형 데이터의 분포가 정규분포를 따르는지의 여부에 대한 검정을 실시하고 정규확률지(Normal Probability Plot)를 도시한다.
Regression
  Regression   최소제곱법에 의한 단순회귀 분석과 중회귀 분석을 실시한다.
 Stepwise   회귀모델에서 변수를 추가하거나 제거하여 적절한 예측변수의 집합을 찾아낸다.
 Best Subsets   주어진 예측 변수들을 가지고 수립할 수 있는 최적의 회귀모델을 찾아낸다.
 Fitted Line Plot   산점도와 함께 적합된 단순 회귀직선, 회귀직선의 신뢰구간 등을 도시해주며, 단순 회귀 결과를 보여준다. 2차, 3차의 곡선 회귀모델을 적합할 수도 있다.
 Residual Plot   잔차 분석을 위한 여러 가지 그래프들을 보여준다.
 Binary Logistic  Regression   이진형의 종속변수에 대한 로지스틱 회귀분석을 실시한다.
 Ordinal Logistic  Regression   서수형의 종속변수에 대한 로지스틱 회귀분석을 실시한다.
 Nominal Logistic  Regression   범주형의 종속변수에 대한 로지스틱 회귀분석을 실시한다.
ANOVA
   One-way   종속변수가 Worksheet내에 하나의 열(列)에 입력되어 있을 때 일원배치 분산분석 실시한다.
 One-way (unstacked)   비교하기 위한 그룹들의 데이터가 Worksheet내에 서로 다른 열(列)에 입력되어 있을 때 일원배치 분산분석 실시한다.
 Two-way   인자가 2개인 경우에 이원배치 분산분석을 실시한다.
 Balanced ANOVA   각 인자의 수준 조합에서 실험한 데이터의 개수가 모두 동일한 균형 설계에 대한 분산분석을 실시한다.
 General Linear Model   일반 선형모형이라고도 불리며, 약어로는 GLM이라고 한다. 다양한 형태의 데이터 분석이 가능하며, 특히 계량형 변수를 공변량 (Covariate) 으로 지정하여 회귀분석과 분산분석을 동시에 수행 가능하다.
 Fully Nested ANOVA   Fully Nested ANOVA를 실시하고 반응변수의 Variance Component를 추정한다.
 Test for Equal Variances   각 인자 수준의 조합에서 실험한 데이터간의 등분산성 여부에 대한 가설 검정을 실시한다.
 Interval Plot   인자의 각 수준 별로 특성치의 평균 값, 평균을 중심으로 표준오차구간 또는 신뢰구간을 도시한다.
 Main Effect Plot   분산분석에서 하나의 인자가 단독으로 반응변수에 미치는 효과인 주효과를 도시한다.
 Interactions Plot   분산분석에서 2개 이상의 인자 사이의 조합으로 인해 발생하는 효과인 교호작용을 도시한다.
DOE
 Factorial
 Create Factorial Design 2수준의 완전요인 실험, 부분요인 실험계획 및 Plackett-Burman 실험을 생성해준다.
Define Custom Factorial Design 이미 Worksheet상에 실험한 데이터가 입력되어 있는 경우 이 데이터들을 요인 배치법의 형태로 정의한다.
Analyze Factorial Design 실험계획에 의하여 설계된 실험순서와 인자수준에 따라 실험을 실시하여 얻어진 데이터의 분석결과를 보여주고, 효과Plot, 잔차 Plot을 보여준다.
Factorial Plot 반응변수와 인자간의 관계를 보여주기 위해 주효과, 교호작용, Cube Plot을 도시한다.
Contour /Surface Plot 반응변수와 2개의 인자간의 관계를 보여주기 위해 Contour Plot, Surface Plot을 도시한다.
Overlaid Contour Plot 여러 반응변수에 대한 각각의 Contour Plot을 하나의 그래프 위에 중첩시켜 도시한다. 동시에 여러 반응변수들을 고려하여 적절한 인자들의 설정을 찾을 수 있도록 한다.
Response Optimizer 하나 이상의 반응변수를 동시에 최적화시킬 수 있는 인자의 설정 조합을 찾을 수 있도록 한다.
 Response Surface Create Response Surface Design 반응표면 실험을 위한 실험배치를 생성한다. 반응표면 실험에는 중심합성(Central Composite) 실험과 Box-Behnken 실험의 2가지가 있다.
Define Custom Response Surface Design 이미 Worksheet상에 실험한 데이터가 있는 경우 이 데이터들을 반응표면 실험의 형태로 정의할 때 사용한다.
Analyze Response Surface Design 반응표면 모델을 사용해 결과를 분석한다. 모델에 1차항 뿐만 아니라 교호작용 항과 2차항을 포함시킬 수 있다.
Contour /Surface Plot 반응변수와 2개의 인자간의 관계를 보여주기 위해 Contour Plot, Surface Plot을 도시한다.
Overlaid Contour Plot 여러 반응변수에 대한 각각의 Contour Plot을 하나의 그래프 위에 중첩시켜 도시한다. 동시에 여러 반응변수들을 고려하여 적절한 인자들의 설정을 찾을 수 있도록 한다.
Response Optimizer 하나 이상의 반응변수를 동시에 최적화시킬 수 있는 인자의 설정 조합을 찾을 수 있다.
Mixture Create Mixture Design 실험 대상 제품이 여러 성분이나 요소로 이루어진 경우에 실시하는 혼합물 실험의 배치를 생성한다.
Define Custom Mixture Design 이미 Worksheet상에 실험한 데이터가 있는 경우 이 데이터들을 혼합물 실험의 형태로 정의할 때 사용한다.
Simplex Design Plot 혼합물 실험배치을 시각적으로 보여주기 위해 인자 수준조합을 삼각형 형태로 배치한다.
Factorial Plot 반응변수와 인자간의 관계를 보여주기 위해 주효과, 교호작용, Cube Plot을 도시한다.
Analyze Mixture Design 혼합물 실험 결과를 분석한다. 모형에 1차, 2차, 3차 항을 포함시킬 수 있다.
Response Trace Plot 혼합물 실험의 각 성분의 효과가 반응변수에 미치는 영향을 시각적으로 보여준다.
Contour /Surface Plot 반응변수의 적절한 수준을 얻기 위한 각 성분의 혼합비율을 찾기 위해 Contour Plot과 Surface Plot을 도시한다.
Overlaid Contour Plot 여러 반응변수에 대한 각각의 Contour Plot을 하나의 그래프 위에 중첩시켜 도시한다. 동시에 여러 반응변수들을 고려하여 적절한 인자들의 설정을 찾을 수 있도록 한다.
Response Optimizer 하나 이상의 반응변수를 동시에 최적화시킬 수 있는 인자의 설정 조합을 찾을 수 있다.
Control Charts
 Define Test   관리도상에서 이상원인을 검출해 내기 위한 Test 조건을 사용자가 지정할 수 있도록 한다.
Box Cox Transformation   데이터가 정규분포를 따르지 않을 때, 특히 한쪽 꼬리가 긴 치우친 형태의 분포를 따르는 데이터를 정규분포로 변환하고자 할 때 사용할 수 있다.
X bar-R   부분군의 평균과 범위를 이용하여 공정평균과 산포를 동시에 관리하기 위한 관리도. Xbar 관리도와 R 관리도를 함께 사용한다.
X bar-S   부분군의 평균과 표준편차를 이용하여 공정평균과 산포를 동시에 관리하기 위한 관리도. Xbar 관리도와 S 관리도를 함께 사용한다.
I-MR   개별 측정치 X에 의하여 공정 평균과 산포를 동시에 관리하기 위한 관리도
I-MR-R/S   Individual 관리도, MR관리도, R관리도 혹은 S관리도를 하나의 화면에 나타내 준다.
Z-MR   측정치 데이터를 표준화하여, 각각의 다른 Run으로부터 수집된 데이터를 하나의 관리도를 통해 해석이 가능하다.
X bar   각 부분군의 평균을 관리도에 타점하여 공정평균을 관리하기 위한 관리도
R   각 부분군의 범위를 관리도에 타점하여 공정의 산포를 관리하기 위한 관리도
S   부분군의 표준편차를 관리도에 타점하여 공정의 산포를 관리하기 위한 관리도
Individual   공정평균을 관리함에 개별 측정치 X를 사용하는 관리도
Moving Range   인접한 두 측정치의 범위인 이동범위 MR을 관리도에 타점하여 공정의 산포를 관리하기 위한 관리도
EWMA   지수가중이동평균 관리도라고 하며, 관리상태에 있는 공정의 미세한 변화를 탐지하기 위해 주로 사용된다.
Moving Average   연속된 데이터들의 이동평균을 사용해 공정 평균의 변화를 관리한다.
CUSUM   목표값과 각 표본 데이터간의 편차에 대한 누적합을 사용해 공정의 미세한 변화를 탐지하기 위해 주로 사용한다.
Zone   공정이 관리상태에 있는지의 여부를 간단하게 평가하기 위해 Zone으로 구분하여 평가하는 관리도
P   불량률 관리도라고도 하며 부분군의 불량률을 이용하여 공정의 불량률을 관리하기 위한 관리도
NP   불량개수 관리도라고도 하며 부분군의 크기가 일정한 경우 불량률을 관리하는 p 관리도 대신 계산상의 편의를 고려하여 부분군에 포함된 불량개수를 관리하기 위한 관리도
C   단위 제품당 결점수를 관리하기 위해 고안된 관리도
U   공정을 단위 크기 당 결점수에 의하여 관리하기 위한 관리도
Quality Tools
 Run chart   데이터를 타점하여 꺽은선 그래프로 나타낸 그림. 이 점들의 형태를 분석하여 데이터의 관리상태 여부가 파악 가능하다.
Pareto chart   불량, 결점, 고장 등의 발생건수나 손실금액을 항목별로 나누어 발생빈도의 순으로 나열하고, 발생빈도의 누적합도 표시한 그림
Cause and Effect   품질특성에 영향을 주는 원인이 어떻게 관계하고 있는가를 한 눈으로 알 수 있도록 작성한 그래프. 생선뼈 그림(Fishbone Diagram) 또는 이시가와 다이아 그램 (Ishikawa Diagram)이라고도 한다.
Capability Analysis (Normal)   히스토그램을 정규분포곡선과 함께 도시해 주고 데이터의 평균, 단기 및 장기 표준편차, 단기 및 장기 공정능력지수, 관측된 PPM, 단기 및 장기 예상 PPM 등을 함께 나타낸다.
Capability Analysis (Between/Within)   장기공정능력지수와 함께 부분 군내의 변동과 부분 군간의 변동을 따로 구하여 이들의 합을 기준으로 계산한 공정능력지수를 보여준다.
Capability Analysis (Weibull)   데이터가 Weibull 분포를 따를 때 공정능력분석을 실시한다.
Capability Sixpack (Normal)   관리도, Dot Plot, 히스토그램, 정규확률지, Capability Plot을 한꺼번에 도시하여 공정의 관리상태 여부, 데이터의 정규성, 공정능력의 부족 여부를 한 눈에 파악할 수 있도록 해 준다.
Capability Sixpack (Weibull)   데이터가 와이블 분포를 따를 때 관리도, 히스토그램, 와이블확률지, Capability Plot 등 다양한 공정능력 분석 결과를 한눈에 파악할 수 있도록 해준다.
Capability Analysis (Binomial)   데이터가 이항 분포를 따를 때 공정능력분석을 실시한다.
Capability Analysis (Poisson)   데이터가 포아송 분포를 따를 때 공정능력분석을 실시한다.
Gage Run Chart   Gage R&R결과 측정시스템이 만족스럽지 못할 경우, 그 원인을 추적하는데 사용할 수 있음
Gage Linearity Study   측정시스템의 정확도에 관련된 특성으로, 측정범위 내에서의 편의의 차이를 말한다
Gage R&R (Crossed)   측정시스템의 정밀도 즉, 반복성과 재현성을 분석하는 도구
Gage R&R (Nested)   Gage R&R(Crossed)와 동일한 분석을 수행하나, 파괴시험과 같이 하나의 부품을 한번만 측정할 수 있고 반복 측정이 불가능한 경우에 사용된다.
Multi-vari chart   다수의 요인들이 특성치에 영향을 미칠 때, 각 요인 별로 특성치에 미치는 효과를 표시한 그래프. 측정값의 요인 별 경향과 관련성을 시각적으로 파악한다.
Symmetry Plot   표본 데이터가 좌우 대칭인 모집단 분포로부터 추출된 것인지 여부를 알고자 할 때 사용되는 그래프
Tables
 Cross Tabulation   Worksheet의 데이터를 평균, 표준편차 등 요약 통계량과 개수 및 비율 등을 보여주는 도표의 형태로 나타내준다. 이산형 변수간의 관계를 검정하기 위해 카이제곱 검정을 수행할 수도 있다.
Chi-Square Test   분할표의 형태로 정리된 이산형 데이터간의 관련성을 파악하기 위해 카이제곱 검정을 수행한다.
Power and Sample Size
 1-Sample Z   1개 모집단에 대한 Z 검정을 실시할 때 검출력(Power), 시료의 크기, 최소의 평균차이를 구할 수 있다.
1-Sample t   1개 모집단에 대한 t 검정을 실시할 때 검출력(Power), 시료의 크기, 최소의 평균차이를 구할 수 있다.
2-Sample t   2개 모집단에 대한 t 검정을 실시할 때 검출력(Power), 시료의 크기, 최소의 평균차이를 구할 수 있다.
1 Proportion   1개 모비율에 대한 검정을 실시할 때 검출력(Power), 시료의 크기, 최소의 비율차이를 구할 수 있다.
2 Proportions   2개 모집단의 모비율 차이에 대한 검정을 실시할 때 검출력(Power), 시료의 크기, 최소의 비율차이를 구할 수 있다.
One-way ANOVA   일원배치 분산분석을 실시할 때, 검출력(Power), 시료의 크기, 평균간의 최대차이를 구할 수 있다.
2 level Factorial Design   2수준 요인배치 실험에서 반복수, 검출력(Power), 최소 효과, 중심점의 수를 구할 수 있다.
